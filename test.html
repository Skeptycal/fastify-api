Fastify-API: A blazing fast REST APIs with Node.js, MongoDB, Fastify and Swagger

A Node.js project
Based on Siegfried Grimbeek's original project on GitHub (tutorial blog post is on Medium).

Build Setup

```bash

install dependencies

npm install

serve with hot reload at localhost:8080

npm start ```

Prerequisites

  • Nodejs
  • MongoDB
